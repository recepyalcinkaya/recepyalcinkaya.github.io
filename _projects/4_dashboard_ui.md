Developed an interactive, touch-enabled LCD interface specifically designed for the vehicle cabin. 

This centralized dashboard acts as the brain of the driver's visual feedback system. It seamlessly streams live data from various sensors (ultrasonic, LiDAR) and displays real-time camera feeds for other ADAS functions. Built to run smoothly in a Linux (Ubuntu) edge environment, it ensures the driver has instant access to critical safety warnings and system statuses.
